# Chunk 1: Core loop (no tools, no fallback, no interrupt)

## Scope

The smallest session that can drive a real LLM turn end-to-end. No tools, no
journal, no interrupt, no permission gates, no fallback model.

## Package split

```
internal/agent/loop/
  loop.go     — Loop{Commands, Events}, New(ctx, sessionID, cfg), listen(), loopStatus, loopState
  config.go   — Config
  command.go  — Command, controlCommand, turnCommand + StartTurn, Shutdown
  event.go    — Event interface + all chunk-1 event types
  turn.go     — runTurn, Turn, turnRunnerStatus

internal/agent/session/
  session.go  — Session{SessionID, *loop.Loop}, New(ctx, cfg)
```

## What to build

### `internal/agent/loop/`

**`loop.go`**

```go
type Loop struct {
    Commands chan<- Command
    Events   <-chan Event
}

func New(ctx context.Context, sessionID string, cfg Config) (*Loop, error)
```

- `Commands` buffered cap 64, `Events` buffered cap 256.
- `New` constructs the gateway via `providers` (no caller injection), starts `listen`
  goroutine, returns immediately.
- `listen` closes `events` on any exit path (defer).

**`listen` — only the pieces needed for Chunk 1:**

First action on entry — before the select loop:
```
emit SessionStarted{SessionID: sessionID}
```

Goroutine-local state, owned exclusively by `listen()` — no mutex needed:

```go
type loopStatus int
const (
    loopIdle        loopStatus = iota // no turn running, accepting new turns
    loopRunning                        // turn is executing
    loopShuttingDown                   // Shutdown received, draining current turn
    loopExited                         // listen() is about to return
)

type loopState struct {
    turnIndex int        // starts at 0; first turn gets index 1
    status    loopStatus // starts at loopIdle
}
```

`internal` channel carries `turnRunnerStatus` directly — no interface wrapper needed in chunk-1:

```go
type turnRunnerStatus int
const (
    turnRunnerStarted turnRunnerStatus = iota // turn began executing
    turnRunnerStopped                          // turn finished normally
    turnRunnerExited                           // goroutine exiting (even on panic)
)
```

`internal` is `chan turnRunnerStatus` (buffered cap 3 — one slot per status value; prevents deadlock when ctx cancels mid-turn and the runner sends started+stopped+exited before listen drains).

Turn runner goroutine:
```
go func():
    defer func():
        recover()                             // catch any panic
        internal <- turnRunnerExited          // always fires, even on panic

    var messages []*llm.AgenticMessage         // owned by this goroutine; grows across turns

    for turn := range turnQueue:
        internal <- turnRunnerStarted
        messages = runTurn(turn.ctx, turn.cmd, turn.turnIndex, messages, cfg, emit)
        internal <- turnRunnerStopped
```

Command routing in `listen`:
```
case cmd := <-commands:
    switch cmd.(type) {
    case controlCommand: handleCommand(cmd, state, ...)
    case turnCommand:    enqueueTurn(cmd, state, ...)
    }
```

`handleCommand` — acts immediately, never queues:
```
func handleCommand(cmd controlCommand, state *loopState, ...):
    switch cmd.(type) {
    case Shutdown:
        if state.status == loopShuttingDown: break
        state.status = loopShuttingDown
        close(turnQueue)
    // Interrupt added in chunk-04
    }
```

`enqueueTurn` — enriches and forwards to turn runner:
```
func enqueueTurn(cmd turnCommand, state *loopState, ...):
    switch startTurn := cmd.(type) {
    case StartTurn:
        if state.status == loopShuttingDown: emit TurnRejected{Reason: "session shutting down"}; return
        select:
        case turnQueue <- Turn{cmd: startTurn, turnIndex: state.turnIndex + 1, ctx: ctx}:
            state.turnIndex++
        default:
            emit TurnRejected{Reason: "turn queue full"}
    }
```

`emit` func — non-blocking send, warn on drop:
```
select:
case events <- ev:
default:
    slog.Warn("event channel full; dropping", "type", fmt.Sprintf("%T", ev))
// no journal in this chunk
```

Exit paths in `listen()` select:
```
case status := <-internal:
    switch status {
    case turnRunnerStarted:
        state.status = loopRunning
    case turnRunnerStopped:
        if state.status != loopShuttingDown {
            state.status = loopIdle
        }
    case turnRunnerExited:
        if state.status == loopShuttingDown:
            state.status = loopExited
            return  // graceful shutdown complete
        state.status = loopExited
        slog.Error("turn runner exited unexpectedly")
        return
    }

case <-ctx.Done():
    // hard kill — only close turnQueue if Shutdown hasn't already closed it
    if state.status != loopShuttingDown {
        close(turnQueue)
    }
    for s := range internal {
        if s == turnRunnerExited { break }
    }
    state.status = loopExited
    return
```

No `runnerDone` channel — `turnRunnerExited` replaces it. The deferred
`recover()` in the turn runner guarantees this fires even on panic.

**`turn.go`**

```go
// Turn is the work item forwarded from the loop goroutine to the turn runner.
// turnIndex and ctx are assigned at enqueue time so runTurn never reads from shared state.
type Turn struct {
    cmd       StartTurn
    turnIndex int
    ctx       context.Context
}
```

```
runTurn(ctx, cmd StartTurn, turnIndex int, messages []*llm.AgenticMessage, cfg Config, emit func(Event)) []*llm.AgenticMessage:
    messages = append(messages, userMessage(cmd.Input))
    emit TurnStarted{TurnIndex: turnIndex}

    req := buildLLMRequest(messages, cfg)
        stream LLM response
            on ctx cancelled: return messages
            on error: emit TurnFailed{TurnIndex: turnIndex, Err: err.Error()}; return messages
        emit TokenDelta per chunk
        assemble complete response via llm.ConcatAgenticMessages
        messages = append(messages, assembled)

        emit TurnDone{TurnIndex: turnIndex, Content: assembled.ContentBlocks}
        return messages
```

**`config.go`**

```go
type Config struct {
    Model   llm.ModelSpec      // Model.System carries the system prompt
    Gateway providers.Settings // BaseURL read from AI_GATEWAY_BASE_URL by the caller
}
```

`New` constructs the token provider and gateway internally:
```
tokens := providers.NewTokenProvider(
    os.Getenv("AI_GATEWAY_TOKEN_URL"),
    os.Getenv("AI_GATEWAY_CLIENT_ID"),
    os.Getenv("AI_GATEWAY_CLIENT_SECRET"),
    os.Getenv("AI_GATEWAY_SCOPE"),
)
// if AI_GATEWAY_TOKEN_URL is empty, use &providers.FixedTokenProvider{} instead
client := providers.NewClient(&cfg.Gateway, tokens)
gw := providers.NewGateway(client, cfg.Gateway, nil, nil)
```

No caller injection of gateway or tokens — only `Config` crosses the boundary.
When the Nexus gateway spec lands, this construction moves to that layer and `Config` gets a `Tokens` field back.

**`command.go`**

```go
// Command is the sealed interface for all commands.
// Only types in this package can implement it (unexported marker method).
type Command interface{ isCommand() }

// controlCommand is a Command that the loop goroutine handles immediately —
// it never enters turnQueue. Shutdown and Interrupt implement this.
type controlCommand interface {
    Command
    isControl()
}

// turnCommand is a Command that the loop goroutine enriches and forwards to
// the turn runner via turnQueue. StartTurn implements this.
type turnCommand interface {
    Command
    isTurn()
}

// StartTurn tells the loop to begin a new turn with the given user input.
// Implements turnCommand — routed to enqueueTurn.
type StartTurn struct {
    Input []*content.Block
}
func (StartTurn) isCommand() {} // satisfies Command
func (StartTurn) isTurn()    {} // satisfies turnCommand → routes to enqueueTurn

// Shutdown tears down the loop gracefully. The current turn (if any) runs to
// completion before the loop exits. Implements controlCommand — handled immediately.
type Shutdown struct{}
func (Shutdown) isCommand() {} // satisfies Command
func (Shutdown) isControl() {} // satisfies controlCommand → routes to handleCommand
```

### `internal/agent/session/`

**`session.go`**

```go
type Session struct {
    SessionID string
    *loop.Loop
}

func New(ctx context.Context, cfg loop.Config) (*Session, error) {
    sessionID := uuid.New().String()
    l, err := loop.New(ctx, sessionID, cfg)
    if err != nil {
        return nil, err
    }
    return &Session{SessionID: sessionID, Loop: l}, nil
}
```

## Tests

Table-driven unit tests in `loop_test.go`:

| Case | Input | Assert |
|---|---|---|
| single turn | one `StartTurn` | `SessionStarted` → `TurnStarted` → ≥1 `TokenDelta` → `TurnDone` |
| two turns serial | two `StartTurn`s | both complete; `TurnIndex` is 1 then 2 |
| turn queue full | 65 `StartTurn`s with blocked turn runner | first 64 accepted (1 running + 63 queued, `turnQueue` cap=63); 65th gets `TurnRejected` |
| shutdown mid-idle | `Shutdown` with no turn running | turn runner exits; `Events` closes |
| shutdown mid-turn | `Shutdown` while turn running | current turn completes then `Events` closes |
| start after shutdown | `StartTurn` after `Shutdown` | `TurnRejected{Reason: "session shutting down"}` |

Use a fake gateway (httptest.Server with canned SSE fixture) — no real network.
